Instalación de MkDocs

Antes de comenzar a crear una página con MkDocs, es necesario instalarlo. Para ello, debemos seguir los siguientes pasos:

  1. Abre una terminal y asegúrate de tener Python 3 instalado.

  2. Instala MkDocs mediante el siguiente comando:

pip install mkdocs

Una vez instalado, podemos proceder a crear nuestro sitio web.
