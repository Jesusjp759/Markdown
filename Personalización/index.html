Personalización del sitio

Ahora que tenemos nuestro sitio creado, podemos personalizarlo para que se ajuste a nuestras necesidades. Estos ajustes se podrán realizar en el archivo mkdocs.yml que se encuentra en la carpeta del sitio creado anteriormente. Algunas de las cosas que podemos personalizar son:

  • El tema: MkDocs viene con varios temas predefinidos que podemos utilizar. Para cambiar el tema, debemos editar el archivo mkdocs.yml y cambiar el valor de theme por el nombre del tema que deseamos utilizar. Por ejemplo:

theme: readthedocs

  • El título: el título que aparece en la parte superior del sitio se puede cambiar editando el archivo mkdocs.yml y cambiando el valor de site_name. Por ejemplo:

site_name: Mi sitio web

  • La descripción: la descripción que aparece debajo del título se puede cambiar editando el archivo mkdocs.yml y cambiando el valor de site_description. Por ejemplo:

site_description: Este es mi sitio web personal

  • La barra de navegación: También se puede añadir una barra de navegación editando el archivo mkdocs.yml usando nav. Por ejemplo:

nav: - Inicio: index.md - Documentación: - Guía de usuario: user_guide.md - Guía de administrador: admin_guide.md - Empezando: getting_started.md
