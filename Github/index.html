Subir la página a GitHub

Para subir la página web creada con MkDocs a GitHub, debemos seguir los siguientes pasos:

  1. Crea un nuevo repositorio en Github.
  2. Abre una terminal y navega hasta la carpeta que contiene la página creada con MkDocs.
  3. Ejecuta el comando ' mkdocs build ' para generar la carpeta con los archivos HTML.
  4. Configura el repositorio remoto como destino de implementación de tu sitio web creando un repositorio git con el comando ' git init ' y sincronizándolo usando el comando ' git remote add origin (url del repositorio) '.
  5. Sube la página usando el comando ' mkdocs gh-deploy '

Una vez completado estos pasos, podrás ver tu página de MkDocs en línea.
