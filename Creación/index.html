Creación de un nuevo sitio

Para crear un nuevo sitio web con MkDocs, debemos seguir los siguientes pasos:

  1. Abre una terminal y navega hasta la carpeta donde deseas crear el sitio.

  2. Crea un nuevo sitio web con el siguiente comando:

mkdocs new my-site

Donde "my-site" es el nombre que le daremos a nuestro sitio. Puedes cambiarlo por cualquier otro nombre que desees.

Una vez creado el sitio, se creará una carpeta con el nombre del sitio y podemos proceder a personalizarlo.
